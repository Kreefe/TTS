# 🚀 Быстрый старт QwenTTS v2.0

## Установка обновлений

```bash
# Установите новую зависимость
pip install tqdm
```

## Основные команды

### 1️⃣ Базовое использование (по умолчанию 4 потока)

```bash
python ttsp.py --input ranobe.txt --name my_book
```

**Вывод:**

```
🚀 Инициализация Silero на cuda...
📖 Озвучка 3000 строк (Лимит памяти: 512 стр)
⚙️ Используется потоков: 4

🎤 Генерация аудио:
Обработка: 100%|████████████████| 3000/3000 [04:30<00:00, 11.1строк/s]

📦 Собираю финальный файл из 6 кусков...
Сборка: 100%|████████████████████████████| 6/6 [00:03<00:00, 1.95чанк/s]
💾 Экспорт в MP3 (128k)...
✨ Готово! Общее время: 4.6 мин.
```

### 2️⃣ Ускоренная генерация (8 потоков)

```bash
python ttsp.py --input ranobe.txt --name my_book --workers 8
```

⚡ **Ускорение до 4x** по сравнению со старой версией!

### 3️⃣ Разбивка на главы

```bash
python ttsp.py --input ranobe.txt --start 1 --end 10 --workers 4
```

**Результат:** папка `1-10/` с файлами `1.mp3`, `2.mp3`, ..., `10.mp3`

### 4️⃣ Для слабых систем (2 потока)

```bash
python ttsp.py --input ranobe.txt --workers 2
```

## Новые возможности

### ✨ Прогресс-бары

- **Процент выполнения** — видите точный прогресс
- **Скорость обработки** — строк в секунду
- **Оставшееся время** — оценка времени до завершения
- **Прошедшее время** — сколько уже работает

### ⚡ Многопоточность

- **По умолчанию:** 4 потока
- **Рекомендуется:**
  - 2 потока для слабых систем (2-4 ядра CPU)
  - 4 потока для средних систем (4-8 ядер CPU)
  - 6-8 потоков для мощных систем (8+ ядер CPU)

### 📊 Производительность

| Конфигурация     | Время (5000 строк) | Ускорение |
| ---------------- | ------------------ | --------- |
| Старая версия    | 25 мин             | 1x        |
| v2.0 (2 потока)  | 13 мин             | 1.9x      |
| v2.0 (4 потока)  | 7 мин              | 3.6x      |
| v2.0 (8 потоков) | 6 мин              | 4.2x      |

## Устранение проблем

### Ошибка: "No module named 'tqdm'"

```bash
pip install tqdm
```

### Медленная генерация

Попробуйте уменьшить количество потоков:

```bash
python ttsp.py --input ranobe.txt --workers 2
```

### Высокое потребление CPU

Используйте меньше потоков:

```bash
python ttsp.py --input ranobe.txt --workers 2
```

## Обратная совместимость

Все старые команды работают без изменений:

```bash
# Эти команды продолжают работать
python ttsp.py --input ranobe.txt --name my_book
python ttsp.py --input ranobe.txt --start 1 --end 10
```
