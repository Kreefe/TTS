# QwenTTS - Text-to-Speech с Silero и Telegram

Проект для преобразования текста в речь с использованием модели Silero TTS и автоматической отправкой результатов в Telegram.

## Описание

Этот проект позволяет конвертировать текстовые файлы (например, книги, ранобэ) в аудиофайлы MP3 с использованием нейросетевой модели синтеза речи Silero TTS. Поддерживается автоматическая отправка готовых аудиофайлов в Telegram через API.

## Основные файлы

- **ttsp.py** - основной скрипт с использованием Telethon для работы с Telegram
- **ttsb.py** - альтернативная версия с использованием Pyrogram
- **parser.py** / **parserv2.py** - парсеры для обработки текстовых файлов
- **ranobe.txt** - входной текстовый файл для конвертации

## Возможности

- 🎙️ Синтез речи с использованием Silero TTS
- 📤 Автоматическая отправка аудио в Telegram
- 📚 Поддержка разбиения на главы
- 🔧 Настраиваемые параметры через аргументы командной строки
- 🌐 Поддержка SOCKS5 прокси

## Использование

### ttsp.py (Telethon версия)

```bash
python ttsp.py --name output_name --input ranobe.txt --start 495 --end 502
```

Параметры:
- `--name` - имя выходного MP3 файла (по умолчанию: final_book)
- `--input` - входной текстовый файл (по умолчанию: ranobe.txt)
- `--start` - начальная глава
- `--end` - конечная глава

### ttsb.py (Pyrogram версия)

```bash
python ttsb.py --name output_name
```

Параметры:
- `--name` - имя выходного MP3 файла (по умолчанию: final_book)

## Требования

- Python 3.7+
- PyTorch
- Silero TTS
- Telethon / Pyrogram
- pydub
- PySocks

## Установка зависимостей

```bash
pip install torch silero pydub telethon pyrogram pysocks
```

## Настройка

Перед использованием необходимо указать свои данные Telegram API в файлах:
- `API_ID` - ID приложения
- `API_HASH` - хеш приложения

Получить можно на https://my.telegram.org/apps

## Структура проекта

```
QwenTTS/
├── ttsp.py              # Основной скрипт (Telethon)
├── ttsb.py              # Альтернативная версия (Pyrogram)
├── parser.py            # Парсер текста
├── parserv2.py          # Парсер текста v2
├── ranobe.txt           # Входной текстовый файл
├── .gitignore           # Исключения для Git

```

## Примечания

- MP3 файлы автоматически исключены из Git (.gitignore)
- Сессионные файлы Telegram также игнорируются
- Поддерживается работа через SOCKS5 прокси
